# Simple NY100/NASDAQ Trading Bot

Un robot de trading especializado para el índice NASDAQ/NY100 en MetaTrader 5.

## 📊 Versión Actual
- **Versión**: v1.6
- **Plataforma**: MetaTrader 5 (MQ5)

## 🎯 Características

- Trading automático para NASDAQ/NY100
- Filtros SMA configurables
- Gestión de riesgo incorporada
- Múltiples configuraciones SET incluidas

## 📁 Estructura del Proyecto

```
├── SimpleNY200_v1.6.mq5      # Código fuente principal
├── Versions/                  # Versiones anteriores
├── Indicator/                # Indicadores personalizados
├── *.set                     # Archivos de configuración
└── *.md                      # Documentación y análisis
```

## 🚀 Cómo Usar

1. Copiar el archivo `.mq5` a la carpeta `MQL5/Experts` de tu MetaTrader 5
2. Compilar en MetaEditor
3. Arrastrar el EA a un gráfico de NASDAQ/NY100
4. Configurar los parámetros según tus necesidades

## 📋 Configuraciones Incluidas

- `SimpleNY200_v1.6_NASDAQ.set` - Configuración NASDAQ estándar
- `SimpleNY200_Default.set` - Configuración por defecto
- `SimpleNY200_SP500.set` - Configuración SP500
