## 🔨 OPCIÓN 1: Compilar desde MetaEditor (Recomendado)

1. **Abrir MetaEditor**
   - En MetaTrader 5 → Menú Tools → MetaQuotes Language Editor
   - O presionar `F4` desde MT5

2. **Abrir el archivo**
   - File → Open → Navegar a: `MQL5/Experts/`
   - Seleccionar: `SimpleNY200_v1.1.mq5`

3. **Compilar**
   - Presionar `F7` o clic en botón "Compile"
   - Verificar en pestaña "Toolbox" que diga:
     ```
     0 error(s), 0 warning(s)
     SimpleNY200_v1.1.ex5 generated
     ```

4. **Verificar archivo compilado**
   - Debe aparecer: `SimpleNY200_v1.1.ex5` en la misma carpeta

---

## 🔨 OPCIÓN 2: Compilar desde Navigator de MT5

1. Abrir MetaTrader 5
2. En panel izquierdo "Navigator" → Expandir "Expert Advisors"
3. Buscar `SimpleNY200_v1.1`
4. Clic derecho → "Compile"
5. Verificar resultado en pestaña "Experts"
